Bluestock Mutual Fund Analysis Capstone
Project Objective

Analyze mutual fund NAV data, calculate performance metrics, perform EDA, build a SQLite database, and create an interactive Power BI dashboard.

Project Structure
data/raw: Original mutual fund NAV files
data/processed: Cleaned datasets and performance metrics
data/db: SQLite database
notebooks: Data ingestion, cleaning, EDA, performance analytics, advanced analytics
scripts: ETL and analytics scripts
sql: Database schema and analytical queries
dashboard: Power BI dashboard
reports: Final report and presentation
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
SQLite
SQLAlchemy
Power BI
Key Outputs
clean_nav_data.csv
performance_metrics.csv
bluestock_mf.db

Final report
